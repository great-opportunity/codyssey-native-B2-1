# 찐친 (Jjinchin)
### 외국인을 위한 체험형 한국 여행 안내 + 한국어 학습 앱

> **"찐(진짜) + 친(친구)"** — 너의 진짜 한국인 친구  
> 외국인이 배우는 첫 한국어 단어가 곧 앱 이름이 되는 구조

---

## 📌 프로젝트 개요

**찐친**은 외국인이 한국 여행 중 '진짜 한국'을 경험할 수 있도록, 체험과 한국어 학습을 하나로 묶은 모바일 앱입니다.  
현지 친구가 옆에서 알려주듯 한국만의 문화·경험을 큐레이션하고, 각 상황에 필요한 한국어 표현을 단계별로 안내합니다.

**한 줄 포지셔닝**  
> 한국인이 진짜 좋아하는 경험을, 그걸 하는 데 필요한 한국어와 함께 안내하는 체험형 한국어 앱.

---

## 🎯 핵심 가치 제안

- 외국인 혼자서는 다가가기 어려운 '진짜 한국'을, 현지 친구가 옆에서 알려주듯 안내
- 경험을 **준비 → 이동 → 현장** 단계로 쪼개고, 각 단계에 필요한 한국어를 붙여줌
- 예약 없이도 콘텐츠를 보고 따라 읽는 것만으로 한국과 한국어에 가까워지는 구조

**포함 스코프:** 경험 큐레이션, 단계별 안내, 상황별 한국어(듣기·따라 읽기·이해), 게임형 진도  
**제외 스코프:** 호텔/교통/액티비티 예약·결제·외부 API 연동

### 화면별 기획 상세

**화면 1 — 홈(Home)**
- 히어로 영상/이미지: 외국인이 골목·시장·찜질방을 즐기는 무드
- 오늘의 찐 경험 추천 카드 1개(크게) + 오늘의 한 마디(한국어 표현 + 듣기 버튼)
- 카테고리 칩: 찐 푸드 / 동네 산책 / 로컬 놀이 / 시장 / 목욕탕·찜질방 / 노래방
- 하단 네비게이션: 홈 · 탐색 · 내 학습 · 프로필

**화면 2 — 경험 탐색(Explorer)**
- 검색창 + 필터 칩(카테고리 / 난이도(혼자 가능도) / 지역 / 필요한 한국어 수준)
- 경험 카드 구성: 썸네일 사진, 경험명, 난이도 배지(🟢 혼자 쉬움 / 🟡 보통 / 🔴 도전), 소요시간·지역, 배우는 표현 5개 뱃지

**화면 2-2 — 경험 상세(Experience Guide)**
1. 커버 영상/사진 + 경험명
2. 공감 스토리: "한국인은 왜 이걸 좋아할까?" 2~3줄
3. 단계별 안내: ① 준비물 ② 가는 법(이동) ③ 현장에서 할 일
4. 한국어 표현 카드(각 단계 인라인): 한글 + 발음(로마자) + 뜻 + 🔊듣기 + 🔍따라 읽기
5. 현장 미션 체크리스트 → 완료 시 뱃지 획득
6. 저장·공유
